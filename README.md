## Explain Version Control
### Version Control also known as source control is the process of managing and tracking changes to changes to files.

## Explain the difference between Git and Github
### Git is a popular version control system used for tracking code changes, who made the chnages and code collaboration while Github is a code hosting platform for version control and collaboration.

## Explain the difference between Git fetch and Git pull
### The Git Fetch command is used to fetch all changes from the remote repository to the local repository. It doesn't make any changes to the current working directory. while Git Pull command is used to fetch all changes from the remote repository to the current working directory. It automatically try to merge or rebase them into our current working directory.

## Explain in simple terms Git cherry-pick and the command for it
### Git cherry-pick is a command that lets you apply a specific commit from one branch onto another without merging the entire branch history. The command for this is "git cherry-pick."
